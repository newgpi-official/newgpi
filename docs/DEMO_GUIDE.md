# NewGPI Demo Guide

Status: Public demo not yet documented

NewGPI will publish an interactive demo only after the experience, data handling and access controls are ready for external users.

## Planned demo flow

The intended public demonstration will show how a user request moves through connectivity, access control, AI orchestration and cloud execution before returning a result to a digital experience.

## What the guide will include at release

1. Demo URL and supported regions
2. Account or guest-access requirements
3. A five-minute guided scenario
4. Expected inputs and outputs
5. Data retention and privacy notes
6. Known limitations
7. Feedback and issue-reporting instructions

## Availability

There is no public demo URL documented in this repository at this time. Do not treat unofficial links or third-party deployments as an official NewGPI demo.

Official updates will be published at https://newgpi.vip and in this repository.
